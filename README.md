# Instrucciones para correr la aplicación cliente-servidor con gRPC

## 1. Correr el servidor

1. Dirígete a la carpeta `GrpcServer`.
2. Ejecuta los siguientes comandos:

    ```bash
    dotnet build
    dotnet run
    ```

## 2. Correr el cliente

1. Dirígete a la carpeta `GrpcClient`.
2. Ejecuta los siguientes comandos:

    ```bash
    dotnet build
    dotnet run
    ```
